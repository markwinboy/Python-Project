Запуск программы сопровождается выводом консоли со следующими командами:
1) add_csv имя папки с csv-файлами/путь до csv-файла - добавляет значения цены в случае задания основной таблицы через команду(main_df), 
	в случае когда не указана основная таблица, то при добавлении создаеться основная на основе добавленных
2) csv_to_excel имя папки с csv-файлами/путь до csv-файла - Перевод csv-файлов(если это папка) в excel файл 
3) excel_to_csv имя папки с excel-файлами/путь до excel-файла - Перевод excel-файлов в csv
4) exit - выход из cmd   
5) main_df путь до excel-файла - создает основную таблицу, можно не указывать в принципе а просто добавлять таблицы
